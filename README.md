# class_materials_sort

講義資料を自動で整理・分類するPythonツールです。

Downloadsフォルダ内の講義資料を読み取り、
PDFテキスト解析を用いて授業ごとに分類し、
デスクトップ上へ整理して出力します。

---

## 主な機能

- 授業ごとの自動分類
- 拡張子ごとのフォルダ整理
- PDFテキスト解析による科目判定
- GUI操作対応
- ログ出力機能
- 重複コピー防止
- 日付条件によるフィルタリング
- 既にコピー済みのファイルを自動スキップ

---

## 使用技術

- Python
- pathlib
- shutil
- tkinter
- pdfplumber

---

## フォルダ構成例

### 整理前

```txt
Downloads/
    pdf/
        数学A 第3回.pdf
        コンピュータ基礎演習 課題.pdf
        知的力学システム1.pdf
```

### 整理後

```txt
Desktop/
    class_materials_sort/
        数学A/
            pdf/
                数学A 第3回.pdf

        コンピュータ基礎演習/
            pdf/
                コンピュータ基礎演習 課題.pdf

        知的力学システム/
            pdf/
                知的力学システム1.pdf
```

---

## GUI

アプリ起動後、
「整理開始」ボタンを押すことで自動整理を開始します。

---

## PDF解析

PDFの1ページ目からテキストを抽出し、
授業名キーワードを検索して分類しています。

使用ライブラリ：

```bash
pip install pdfplumber
```

---

## 実行方法

```bash
python main.py
```

---

## ログ機能

整理結果は `log.txt` に保存されます。

例：

```txt
数学A 第3回.pdf を 数学A/pdf にコピーしました。
知的力学システム1.pdf は既にコピー済みです。
```

---

## 重複ファイル対策

既に同名ファイルが存在する場合は、
再コピーせずスキップします。
